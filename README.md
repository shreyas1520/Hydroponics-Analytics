# 📊 Hydroponics Monitoring Analytics Dashboard

## 📌 Project Overview

This project demonstrates an end-to-end data analytics workflow for a smart hydroponics monitoring system. Sensor data collected from an IoT-based hydroponics setup is cleaned, stored, analyzed, and visualized to generate operational and business insights.

The project covers the complete analytics lifecycle including data preprocessing, SQL analysis, KPI development, and interactive Power BI dashboards.

---

## 🎯 Objectives

- Analyze hydroponics sensor data
- Monitor environmental conditions
- Track pump and actuator activity
- Identify abnormal operating conditions
- Generate business insights
- Build an executive dashboard for decision-making

## 📈 Key Business Insights

- Average pH remained close to the ideal hydroponic range.
- Water level followed cyclic refill patterns.
- High temperature and low TDS generated the highest number of alerts.
- Pump activity increased during abnormal environmental conditions.
- Environmental trends were monitored using interactive Power BI dashboards.
